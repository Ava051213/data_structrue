# 运算符、表达式和语句

## 3.1 算术运算符和算术表达式

### 算术运算符类型
- **基本运算符**：
  - 加(`+`)、减(`-`)
  - 乘(`*`)、除(`/`)、求余(`%`)
  
- **自增/自减运算符**：
  - `++` (自增)
  - `--` (自减)
  - 操作元必须是整型或浮点型变量（不能是常量或表达式）

### 算术表达式
- 按运算符两边的操作元的**最高精度**保留结果精度

## 3.2 关系运算符和关系表达式

### 关系运算符
- 运算结果为boolean型(`true`/`false`)
- **大小关系**：`>`, `<`, `>=`, `<=`
- **等/不等**：`==`, `!=`

## 3.3 逻辑运算符和逻辑表达式

### 逻辑运算符类型
| 运算符    | 描述  | 特点   |
| ------ | --- | ---- |
| `&&`   | 逻辑与 | 短路求值 |
| &#124&#124 | 逻辑或 | 短路求值 |
| `!`    | 逻辑非 |      |

## 3.4 赋值运算符和赋值表达式
- 左操作元必须是**变量**
- 复合赋值运算符：`+=`, `-=`, `*=`, `/=`, `%=`

## 3.5 移位运算符

### 移位操作类型
| 运算符 | 示例 | 说明 |
|--------|------|------|
| `<<` | `a << n` | 左移，低位补0 |
| `>>` | `a >> n` | 右移，高位补符号位 |

## 3.6 位运算符

### 位操作类型
| 运算符  | 描述   |
| ---- | ---- |
| `&`  | 按位与  |
| &#124 | 按位或  |
| `~`  | 按位非  |
| `^`  | 按位异或 |

## 3.7 条件运算符
- 三目运算符：`op1 ? op2 : op3`
- `op1`必须是boolean型

## 3.8 instanceof运算符
- 语法：`对象 instanceof 类`
- 返回boolean值，判断对象是否为指定类的实例

## 3.9 一般表达式
- Java表达式：用运算符及操作元连接起来的符合Java规则的式子

## 3.10 语句概述

### 语句分类
1. 方法调用语句
2. 表达式语句
3. 复合语句（代码块）
4. 控制语句：
   - 条件分支
   - 循环
   - 跳转
5. package/import语句

## 3.11 分支语句

### 分支结构类型
- if-else条件分支
- switch开关语句：
  - 表达式值必须是整型或字符型
  - `default`分支可选

## 3.12 循环语句

### 循环结构类型
| 类型 | 语法 | 特点 |
|------|------|------|
| while | `while(条件){...}` | 先判断后执行 |
| do-while | `do{...}while(条件)` | 至少执行一次 |
| for | `for(init;cond;update){...}` | 包括增强for循环 |

## 3.13 跳转语句
- `break`：中断循环
- `continue`：跳过本次循环
