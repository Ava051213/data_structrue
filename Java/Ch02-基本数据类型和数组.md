# 基本数据类型和数组

## 2.1 标识符和关键字

### 标识符
- 定义：用于标识类名、变量名、方法名等的有效字符序列
- 组成规则：
  - 由字母、下划线、美元符号$和数字组成
  - 第一个字符不能是数字
  - 长度不受限制
- 限制：
  - 不能使用true、false、null（这些是常量而非关键字）

### 关键字
- Java语言中已被赋予特定意义的单词
- 不可用作标识符名称

## 2.2 基本数据类型

### 8种基本数据类型分类
| 类型 | 包含的数据类型 |
|------|----------------|
| 逻辑类型 | boolean |
| 字符类型 | char |
| 整数类型 | byte, short, int, long |
| 浮点类型 | float, double |

### 详细说明
1. **boolean类型**
   - 常量：true/false

2. **char类型**
   - 特点：
     - 使用Unicode字符
     - 最高位不是符号位（无负数）
   - 注意事项：
     - 查看Unicode顺序位置必须使用int类型转换

3. **整数类型**
   - 常量表示：
     - 十进制：123
     - 八进制：077
     - 十六进制：0x3ABC
   - 位数：
     ```mermaid
     graph LR
     A[byte-8位] --> B[short-16位] --> C[int-32位] --> D[long-64位]
     ```

4. **浮点类型**
   - float/double的区别

## 2.3 类型转换

### 自动转换规则
byte → short → int → long → float → double

### 强制转换
- 高级别转低级别需显式转换
- 示例：`int a = (int)3.14;`

## 2.4 输入输出

### 输出格式
| 格式符 | 说明 |
|--------|------|
| %d | 整型 |
| %c | 字符 |
| %f | 浮点(6位小数) |
| %s | 字符串 |
| %md | m列宽整型 |
| %m.nf | m列宽,n位小数 |

### 输入方法
```java
Scanner input = new Scanner(System.in);
// 常用方法：
input.nextInt()   // 读取int
input.nextDouble() // 读取double
input.nextLine()  // 读取字符串
```

## 2.5 数组

### 基本概念
- 相同类型数据的顺序集合
- 下标从0开始

### 声明与创建
```java
// 一维数组
int[] arr1;  // 声明
arr1 = new int[5];  // 创建

// 二维数组
double[][] arr2 = new double[3][4];
```

### 注意事项
1. 声明时不能指定元素个数
2. 使用前必须分配内存空间
3. 数组越界会抛出`ArrayIndexOutOfBoundsException`

### 工具方法
1. **System.arraycopy()**
   ```java
   arraycopy(src, srcPos, dest, desPos, length)
   ```
2. **Arrays类**
   [官方文档](https://docs.oracle.com/en/java/javase/16/docs/api/java.base/java/util/Arrays.html)
